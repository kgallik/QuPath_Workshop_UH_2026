# Adding the P drive

If the P drive is not visible on your computer, use these instructions to add it.

Open file explorer, click on the `...` and then `Add a network location`:

![Pdrive1](/Tutorials/PNGs/P_Drive1.png)

In the new window, enter in the following information:

![Pdrive2](/Tutorials/PNGs/P_Drive2.png)

When you click on `My PC` in File Explorer, you should now see the P Drive:

![Pdrive3](/Tutorials/PNGs/P_Drive3.png)
