# Helpful Shortcuts

`Ctrl + L` to open search menu to find tools

`Crtl + Shift + A` to create a full image rectangle

Tool shortcuts:

- `M` mouse/panning (also holding space bar for temporary panning)
- `R` rectangle tool
- `O` oval tool
- `L` line tool
- `P` closed polygon tool
- `V` open polyline tool
- `B` brush tool
- `W` wand tool
- `.` points tool

`Shift + S` Selection mode on/off

`Shift + C` brightness and contrast, channel visibility

`A` show/hide annotations

`Shift + F` fill annotations on/off

`N` show/hide annotations names

`G` show/hide TMA grid

`D` show/hide dections

`F` fill detections on/off

`C` show/hide preview overlay (slider controls opacity of all overlayed objects)
